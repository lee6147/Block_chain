MetaMask 지갑과 Remix IDE를 이용하여 ERC20 토큰을 실제 테스트넷에 배포합니다. 순서대로 따라해 주세요.

1

Remix IDE 접속

브라우저에서 Remix IDE에 접속합니다. 별도 설치 없이 웹 브라우저만으로 스마트 컨트랙트를 개발하고 배포할 수 있습니다.

Remix IDE 열기 (새 탭)

팁: Chrome 또는 Firefox 브라우저를 사용하세요. MetaMask 확장 프로그램이 설치되어 있어야 합니다.

2

새 파일 생성 → MyToken.sol 코드 붙여넣기

  1. 1. 좌측 파일 탐색기에서 contracts 폴더를 우클릭합니다.
  2. 2. New File을 클릭하고 파일명을 MyToken.sol로 입력합니다.
  3. 3. 아래 코드를 전체 복사하여 편집기에 붙여넣습니다. 
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.20;

import "@openzeppelin/contracts/token/ERC20/ERC20.sol";

contract MyToken is ERC20 {
    constructor(uint256 initialSupply) ERC20("My Token", "MTK") {
        _mint(msg.sender, initialSupply * 10 ** decimals());
    }
}

주의: Remix는 기본적으로 OpenZeppelin 라이브러리를 자동으로 불러올 수 있습니다. 만약 import 오류가 발생하면 DEPS → Install 버튼을 클릭하거나, 좌측 Plugin Manager에서 DEPS 플러그인을 활성화하세요.

3

컴파일러 설정

  1. 1. 좌측 사이드바에서 Solidity Compiler 아이콘(▲ 모양)을 클릭합니다.
  2. 2. COMPILER 드롭다운에서 0.8.20 이상 버전을 선택합니다.
  3. 3. Compile MyToken.sol 버튼을 클릭합니다.
  4. 4. 컴파일 성공 시 아이콘에 초록색 체크 표시가 나타납니다.

팁: Auto compile 체크박스를 활성화하면 코드 수정 시 자동으로 컴파일됩니다. Enable optimization도 체크하면 가스 비용을 줄일 수 있습니다.

4

배포 환경: Injected Provider - MetaMask 선택

  1. 1. 좌측 사이드바에서 Deploy & Run Transactions 아이콘(▶ 모양)을 클릭합니다.
  2. 2. ENVIRONMENT 드롭다운에서 Injected Provider - MetaMask를 선택합니다.
  3. 3. MetaMask 팝업이 열리면 연결을 클릭합니다.
  4. 4. MetaMask에서 테스트넷(Sepolia 등)이 선택되어 있는지 확인합니다.

주의: 실제 이더(ETH)를 사용하지 않으려면 반드시 테스트넷(Sepolia)을 사용하세요. 테스트넷 ETH는 SepoliaFaucet에서 무료로 받을 수 있습니다.

5

생성자 인자 입력 → Deploy

  1. 1. CONTRACT 드롭다운에서 MyToken이 선택되어 있는지 확인합니다.
  2. 2. Deploy 버튼 옆 입력란에 초기 발행량을 입력합니다. 1000 입력 시 1,000 MTK가 발행됩니다.
  3. 3. Deploy 버튼을 클릭합니다.
  4. 4. MetaMask 팝업에서 가스 비용을 확인하고 확인을 클릭합니다.

// 생성자 인자 예시

initialSupply: 1000

// → 배포자 지갑에 1000 MTK (1000 * 10^18 wei) 발행

팁: 트랜잭션이 제출되면 MetaMask에서 Activity 탭에서 진행 상황을 확인할 수 있습니다. 보통 10~30초 내에 완료됩니다.

6

배포된 컨트랙트 주소 확인

  1. 1. Remix 화면 하단 콘솔(Terminal)에서 트랜잭션 결과를 확인합니다.
  2. 2. 배포 성공 후 Deployed Contracts 섹션에 컨트랙트가 나타납니다.
  3. 3. 컨트랙트 이름 옆 복사 아이콘을 클릭하여 주소를 복사합니다.
  4. 4. 아래 저장 폼에 붙여넣기 하세요.

주의: 컨트랙트 주소는 0x로 시작하는 42자리 16진수입니다. 주소를 잘못 저장하면 토큰을 다시 찾을 수 없으니 반드시 정확히 복사하세요.

7

MetaMask에서 "토큰 가져오기" → 주소 입력

  1. 1. MetaMask를 열고 Tokens 탭으로 이동합니다.
  2. 2. 하단의 토큰 가져오기(Import tokens)를 클릭합니다.
  3. 3. 토큰 계약 주소 필드에 6단계에서 복사한 컨트랙트 주소를 붙여넣습니다.
  4. 4. 토큰 심볼(MTK)과 소수점(18)이 자동으로 입력됩니다.
  5. 5. 토큰 추가를 클릭하면 MetaMask 지갑에서 MTK 잔액을 확인할 수 있습니다.

팁: 토큰이 자동으로 감지되지 않으면 Custom Token 탭에서 컨트랙트 주소를 직접 입력하세요.

Hardhat / Foundry 배포 (로컬 개발 도구)

Remix IDE 외에도 로컬 개발 환경에서 스크립트를 통해 배포할 수 있습니다. 프로젝트 규모가 커지면 아래 도구들을 사용하는 것이 권장됩니다.

Hardhat

# 프로젝트 초기화
npm init -y
npm install --save-dev hardhat
npx hardhat init

# 의존성 설치
npm install @openzeppelin/contracts

# Sepolia 배포
npx hardhat run scripts/deploy.js \
  --network sepolia

Foundry

# Foundry 설치
curl -L https://foundry.paradigm.xyz | bash
foundryup

# 프로젝트 초기화
forge init my-token
cd my-token

# Sepolia 배포
forge script script/Deploy.s.sol \
  --rpc-url $SEPOLIA_RPC \
  --broadcast --verify
