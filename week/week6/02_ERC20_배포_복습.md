# 02. 본인 ERC20 배포 복습

Week 5에서 다룬 ERC20 배포를 다시 한 번 짚고 넘어간다.
본인이 배포한 ERC20을 스테이킹 담보로 써도 되고, 조교가 배포한 HNL을 그대로 써도 된다.
여유가 있다면 **본인 토큰을 직접 배포**하는 쪽이 학습에 좋다.

## 레퍼런스 컨트랙트

`20_ExampleToken.sol` — OpenZeppelin `ERC20` + `Ownable`.

```solidity
constructor(
    string memory name_,
    string memory symbol_,
    uint256 initialSupply
) ERC20(name_, symbol_) Ownable(msg.sender) {
    _mint(msg.sender, initialSupply);
}
```

## Remix 배포 순서

1. Remix → `20_ExampleToken.sol` 열기.
2. **Solidity Compiler** 탭:
   - Compiler: `0.8.24` (또는 `0.8.x` 최신)
   - Auto compile 켜기 → 에러 없는지 확인.
3. **Deploy & Run** 탭:
   - Environment: **Injected Provider - MetaMask** (MetaMask는 Sepolia 선택)
   - Contract: `ExampleToken`
   - 생성자 인자 입력 예:
     - `name_`: `"MyStakeCoin"`
     - `symbol_`: `"MSC"`
     - `initialSupply`: `1000000000000000000000000` (= 1,000,000 토큰, decimals=18)
4. **Deploy** 클릭 → MetaMask 서명.
5. 배포 완료 후 Etherscan에서 컨트랙트 주소 확인 → **MetaMask에 Import tokens** 하여 잔액 확인.

## 주의

- `initialSupply`는 **wei 단위**(18 decimals). 100만 토큰 = `1000000 * 1e18`.
- Sepolia 가스가 부족하면 먼저 [01번 문서](./01_Faucet_수령_가이드.md)에서 ETH 수령.
- `Ownable(msg.sender)` — OpenZeppelin v5부터 생성자 파라미터로 owner를 지정. 복사해 쓸 때 버전 확인.

## 배포 후 기록

제출 양식에 **본인 토큰 주소**를 남겨야 한다. Etherscan 링크도 같이 기록:

```
TOKEN_ADDRESS = 0x...
https://sepolia.etherscan.io/address/0x...
```
