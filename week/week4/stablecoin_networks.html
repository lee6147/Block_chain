Section 01
왜 스테이블코인 "전용" 블록체인이 필요한가?

기존 방식의 문제

이더리움에서 USDT 10달러를 보내려면?

  • USDT(보내고 싶은 돈) + ETH(가스비) 둘 다 보유해야 함
  • ETH 가격이 하루에 10~20% 변동 → 가스비 예측 불가
  • NFT 민팅, DeFi 등 다른 트랜잭션과 경쟁 → 혼잡 시 가스비 폭등
  • 최종 확인까지 수분~수십분 소요

트랜잭션 흐름 시뮬레이션

버튼을 눌러 기존 방식과 전용 네트워크의 차이를 비교해보세요.

Section 02
핵심 개념 복습

스테이블코인(Stablecoin)

법정화폐에 가치가 연동(페깅)되어 가격이 안정적인 암호화폐

USD USDT, USDC → 1토큰 = 1달러
KRW BaseCoin, OKRW → 1토큰 = 1원
EUR EURC → 1토큰 = 1유로

Gas Fee & Gas Token

Gas Fee = 블록체인에서 트랜잭션 처리를 위해 검증자에게 지불하는 수수료

Gas Token = 가스비 지불에 사용하는 토큰

이더리움 → Gas Token = ETH (가격 변동 큼)
Plasma → Gas Token = XPL (제로 수수료!)
StableNet → Gas Token = BaseCoin (1원 고정, 안정적)

L1 vs L2 네트워크 구조

Layer 2 (L2) — 확장 솔루션
Arbitrum, Base, Optimism, zkSync...

↓ 보안 의존 ↓

이더리움, 비트코인, Plasma, StableNet...
독립 운영 / 자체 합의 메커니즘 / 기반 보안 제공

Section 04
StableNet 심층 분석

왜 StableNet을 특별히 다루나요?

교수님께서 한국의 코인 회사 "위메이드(WeMade)"를 언급하시며, 학생들이 위메이드가 만든 StableNet을 직접 조사해보길 권하셨습니다.

6개 L1 스테이블코인 전용 네트워크 중, StableNet은 한국 기업이 만든 KRW(원화) 기반 네트워크라는 점에서 우리에게 가장 가까운 사례입니다. 글로벌 트렌드(Tether→Plasma, Circle→Arc, Stripe→Tempo)와 한국 기업(WeMade→StableNet, Hashed→Maroo)의 움직임을 비교하면 이 시장의 큰 그림을 이해할 수 있습니다.

위메이드(WeMade)는 어떤 회사?

원래 한국의 대표 게임 회사 (대표작: 미르의 전설, 미르4). 블록체인으로 사업을 적극 확장하며 WEMIX 생태계를 구축했습니다.

WEMIX 1.0

클레이튼(Klaytn) 기반 사이드체인으로 블록체인 게임 생태계 시작

WEMIX 3.0 (2022~)

자체 L1 메인넷으로 전환. SPoA 합의, 40개 Node Council Partner, EVM 호환

WEMIX 생태계 확장

WEMIX.Fi(DeFi) + NILE(NFT) + WEMIX Play(게임) + WEMIX$(달러 스테이블코인)

StableNet (현재 Pre-launch)

스테이블코인 결제 전용 별도 L1 구축. BaseCoin(KRW 페깅)을 가스 토큰으로 사용

BaseCoin이 특별한 이유

대부분의 블록체인: 가스비 = 자체 토큰(ETH, MATIC 등) → 토큰 가격 변동으로 수수료 예측 불가

StableNet: 가스비 = BaseCoin (1원 고정) → "이 트랜잭션에 약 10원 소요" 처럼 원화 단위로 정확히 예측 가능

실생활 결제 서비스나 기업 도입에 매우 유리합니다. 커피숍에서 블록체인 결제를 한다고 할 때, 수수료가 100원인지 10,000원인지 모르면 사용할 수 없겠죠?

Q1. 스테이블코인 전용 블록체인이 필요한 가장 핵심적인 이유는?

블록체인 기술을 홍보하기 위해
가스비를 스테이블코인으로 내서 수수료를 예측 가능하게 만들기 위해
이더리움이 너무 오래되었기 때문에
비트코인의 가격을 올리기 위해
기존 블록체인(이더리움)은 ETH로 가스비를 내야 하고, ETH 가격 변동이 커서 수수료 예측이 불가합니다. 스테이블코인 전용 네트워크는 안정적인 스테이블코인으로 가스비를 내므로 수수료가 예측 가능합니다.

Q2. StableNet의 Gas Token인 BaseCoin은 무엇에 페깅되어 있나요?

미국 달러 (USD)
한국 원화 (KRW)
유로 (EUR)
비트코인 (BTC)
BaseCoin은 KRW(한국 원화)에 페깅된 스테이블코인입니다. 1 BaseCoin = 1원이므로, 수수료를 원화 단위로 바로 계산할 수 있습니다.

이더리움 (Ethereum)
비트코인 (Bitcoin)
솔라나 (Solana)

Plasma는 비트코인의 사이드체인으로 동작합니다. Modified Reth로 EVM 호환성을 제공하지만, 보안 앵커는 이더리움이 아닌 비트코인입니다.

Q5. Maroo(Hashed)의 이중 경로 트랜잭션 모델에서, 규제 준수가 필요한 경로는?

Maroo의 Regulated Path는 PCL(Programmable Compliance Layer)을 통해 신원인증, 법적 관할 준수, 자산 유형 규칙, 거래 한도를 사전에 검증합니다. Open Path는 개발자용으로 무허가 실험이 가능합니다.

Q6. Optimistic Rollup과 ZK Rollup의 가장 큰 차이는?
