⚠ 주의사항 & 트러블슈팅

자주 발생하는 오류와 해결 방법을 순서대로 정리했습니다.

▶

1. HTML 파일을 직접 열면 MetaMask가 작동하지 않음

증상

index.html을 더블클릭해서 열면 주소가 file:///C:/...로 시작하고, MetaMask가 "감지되지 않음"으로 표시된다.

원인

MetaMask는 보안상 file:// 프로토콜에서는 window.ethereum을 주입하지 않는다. 반드시 http:// 또는 https://로 접근해야 한다.

해결

start.bat을 더블클릭하여 로컬 서버(http://localhost:5500)로 실행한다.

▶

2. "MetaMask가 감지되지 않았습니다" 표시

증상

localhost로 접속했는데도 "MetaMask가 감지되지 않았습니다"가 표시된다.

원인

MetaMask 확장 프로그램이 비활성화되어 있거나, 잠금 상태(비밀번호 미입력)이면 window.ethereum이 주입되지 않을 수 있다.

해결 (3단계)

  1. chrome://extensions에서 MetaMask가 활성화되어 있는지 확인
  2. 브라우저 툴바의 🦊 아이콘을 클릭하여 MetaMask 잠금 해제 (비밀번호 입력)
  3. 페이지 새로고침(F5) 또는 "다시 확인" 버튼 클릭
▶

3. "지갑 연결" 클릭 후 아무 반응 없음

증상

"지갑 연결" 버튼을 눌렀는데 MetaMask 팝업이 나타나지 않거나, "연결 중..."에서 멈춘다.

원인

이전에 요청한 연결 팝업이 아직 대기 중일 수 있다. MetaMask는 한 번에 하나의 연결 요청만 처리하므로, 이전 팝업을 처리하지 않으면 새 요청이 무시된다.

해결

브라우저 툴바의 🦊 MetaMask 아이콘을 클릭하면 대기 중인 팝업이 보인다. 거기서 "연결" 또는 "거부"를 선택하면 된다.

▶

4. "Sepolia 테스트넷으로 전환해주세요" 경고

증상

지갑 연결에 성공했지만 노란색 경고 배너가 표시된다.

원인

현재 MetaMask가 Ethereum Mainnet 등 다른 네트워크에 연결되어 있다. 이 앱은 Sepolia 테스트넷(Chain ID: 11155111)에서 동작하도록 설계되었다.

해결

경고 배너 옆의 "Sepolia 전환" 버튼을 클릭하면 자동으로 네트워크가 전환된다.

▶

5. 가격 데이터를 가져올 수 없음

증상

가격 테이블 대신 "가격 데이터를 가져올 수 없습니다" 에러가 표시된다.

원인

인터넷 연결 불안정이거나, CoinGecko API의 무료 호출 제한(분당 10~30회)에 걸렸을 수 있다. CORS 정책으로 직접 호출이 차단될 수도 있다.

해결

인터넷 연결을 확인하고, "다시 시도" 버튼을 클릭한다. 앱이 자동으로 3개의 우회 경로(CORS 프록시)를 순차 시도한다.

2주차 핵심 개념

▶

EOA vs 스마트 컨트랙트 계정

EOA (Externally Owned Account): 개인키로 제어하는 일반 지갑. MetaMask가 만드는 것.

CA (Contract Account): 코드로 제어되는 계정. 스마트 컨트랙트 배포 시 생성.

비유

EOA = 개인 통장 (본인 서명으로 출금), CA = 자동이체 규칙이 담긴 통장 (조건 충족 시 자동 실행)

정석 설명

EOA는 ECDSA(secp256k1) 개인키-공개키 쌍으로 제어된다. CA는 EVM 바이트코드가 저장된 계정으로, 다른 계정의 메시지 호출에 의해서만 코드가 실행된다.

▶

시드 문구 (Seed Phrase)

12~24개의 영단어로 된 마스터 키. 이것만 있으면 지갑을 완전히 복구 가능.

절대 타인에게 공개 금지 — 시드 문구 = 모든 자산의 소유권.

비유

시드 문구 = 금고의 마스터키. 복사해서 누군가에게 주면 그 사람이 금고를 열 수 있음.

정석 설명

BIP-39 표준에 따라 128~256비트 엔트로피에서 생성된 니모닉 단어 목록. BIP-32/44 경로에 따라 무한한 개인키를 파생할 수 있는 마스터 시드 역할을 한다.

▶

Sepolia 테스트넷

이더리움 메인넷과 동일하게 동작하지만, ETH가 무료인 연습용 네트워크.

Chain ID: 11155111 (0xaa36a7)

비유

테스트넷 = 보드게임 화폐. 규칙은 진짜와 같지만 실제 가치 없음. 실수해도 돈을 잃지 않는다.

정석 설명

Sepolia는 이더리움의 PoS(Proof of Stake) 기반 테스트 네트워크다. EIP-155의 Chain ID(11155111)로 식별되며, 트랜잭션 서명 시 replay attack을 방지한다.

입문자 추천도: ★★★★★

🔷 Ethereum (ETH)

시가총액 2위, 스마트 컨트랙트의 원조. DeFi, NFT 등 생태계의 기반.

입문자 추천도: ★★★★★

3 MetaMask 첫 설정 순서

  1. 설치 — metamask.io에서 Chrome 확장 프로그램 설치
  2. 지갑 생성 — "새 지갑 만들기" 선택 → 비밀번호 설정
  3. 시드 문구 백업 — 12개 영단어를 종이에 적어서 안전한 곳에 보관
  4. 테스트넷 활성화 — 설정 > 고급 > "테스트 네트워크 표시" 켜기
  5. Sepolia 선택 — 네트워크 드롭다운에서 "Sepolia 테스트 네트워크" 선택
  6. 테스트 ETH 수령 — Faucet에서 무료 ETH 받기
